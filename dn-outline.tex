\section{.NET 框架简介}
\begin{frame} \frametitle{什么是 Microsoft .NET} 

  % Microsoft .NET 是 Microsoft XML Web services 平台。XML Web services 允许应
  % 用程序通过 Internet 进行通讯和共享数据，而不管所采用的是哪种操作系统、设备
  % 或编程语言。Microsoft .NET 平台提供创建 XML Web services 并将这些服务集成在
  % 一起之所需。

  \begin{itemize}
    \item .NET Framework
    \item Visual Studio
  \end{itemize}
\end{frame}

\begin{frame} \frametitle{Microsoft .NET 平台}
  \begin{itemize}
  \item 底层操作系统 -- Windows
  \item 辅助产品 -- .NET 企业服务器
  \item 开发以及运行平台 -- .NET框架
  \item 继承开发环境 -- Visual Studio .NET
  \end{itemize}
\end{frame}


\begin{frame} \frametitle{Microsoft .NET框架}
  \begin{itemize}
  \item .NET 框架是创建 Windows 应用程序的基础
  \item 简化了应用程序的开发 \\
    ASP.NET, Windows Forms, ASP.NET Web Services, Enterprise Services, ADO.NET
  \item .NET 类库提供开发所需的各种基本功能库
  \item .NET 类库可以支持各种类型的应用程序
  \item 简化开发，屏蔽底层细节，统一的编成模式
  \item 使用国际标准，如XML、SOAP、XPath等
  \item 容易维护、部署等

  \end{itemize}

\end{frame}
